### Compiling Programs Using XDUI

To compile programs using the XDUI library, follow these steps:

1. Open your project in Visual Studio
2. In Solution Explorer, right-click on the VC++ project file
3. Select "Properties"
4. Navigate to Linker -> System
5. Set SubSystem to "Windows (/SUBSYSTEM:WINDOWS)"
6. Compile normally

This configuration ensures that your application will run as a Windows GUI application without showing a console window.

### 使用XDUI库编写的程序编译方法

要编译使用XDUI库的程序，请按照以下步骤操作：

1. 在Visual Studio中打开您的项目
2. 在解决方案资源管理器中，右键单击VC++项目文件
3. 选择"属性"
4. 导航到链接器 -> 系统
5. 将子系统设置为"窗口(/SUBSYSTEM:WINDOWS)"
6. 正常编译

此配置确保您的应用程序将作为Windows GUI应用程序运行，而不会显示控制台窗口。
